SCENE 06 · 노정석 사고 체계

6개의 인지 패턴 식별

인터뷰어의 사고법 = 관찰 프레임워크

PATTERN 01

임계점을 찍게 만든다

모호한 감탄을 허용하지 않음. "정확히 언제?", "어떤 도구?" 해상도를 강제

PATTERN 02

기존 제도를 올려놓고 부순다

"이게 대학을 바꾸는 거야?" 기능-형태 분해 → 대체재 식별

PATTERN 03

긴 담론을 한 문장으로 압축

압축이 틀리면 정정, 맞면 확정. compression-error 검증

PATTERN 04

기술 → 인간 조건으로 강제 승격

기술 이야기 → "인간에게 무엇을 의미하는가?" → 제도 설계로 전환

PATTERN 05

자기 경험을 투입해 마찰 생성

"제 조카도 명문대에..." 이론과 현실의 충돌 지점에서 통찰 추출

PATTERN 06

대화 구조 자체에 대한 발언

"닫지 못한 괄호가 많다" 사유 과정의 투명성. 권력을 숨기지 않음

SCENE 07 · 응용 — 김서준 분석

같은 방법론으로 김서준 분석

스킬 테스트: 패턴 추출 → 검증 → 압축

PATTERN 01

개인 체험 → 보편 원리 승격

구체적 일화에서 시작 → 구조적 결론으로. 체화된 관찰만이 사유의 출발

PATTERN 02

이원 대조로 인지 전환

도구 vs 주체, 양 vs 질, 유틸리티 vs 인스퍼레이션. 비연속성 식별

PATTERN 03

역사적 궤적 위에 배치

400년 단위, 인류사 단위. 현재가 아니라 방향을 본다

PATTERN 04

극단으로 밀어붙이기

"다 없어지면 뭐가 남나?" 극단의 사유로 제거 불가능한 핵심 추출

PATTERN 05

구체적 숫자로 고정

30/20, 4시간, 30%/70%, 20/1000. 숫자 없으면 관측이 안 끝난 것

PATTERN 06

"재밌다"를 전환 신호로 사용

놀람 = 관측 데이터. 감정이 아니라 구조적 전환의 탐지기

✅ 패턴 검증 완료: 반례 탐색, 패턴 간 긴장 식별, 한 문장 압축까지

SCENE 09 · 스킬 요약

discourse-pattern-extractor

대화에서 사고의 알고리즘을 추출하는 5단계

화자의 질문·개입·메타 발언을 모두 추출한 뒤, 반복되는 인지 행위를 패턴으로 식별하고, 반례 탐색으로 검증하여, 가치 체계를 종합한다.

01
발화 추출
질문, 개입, 메타 발언을 맥락과 함께 분리
02
패턴 식별
반복되는 인지 행위를 군집화 · 명명 · 구조화
03
패턴 검증
반례 탐색, 패턴 간 충돌, 변화 추이 분석
04
가치 체계
말이 아닌 행위에서 가치 추출 · 계층화
05
한 문장 압축
전체 분석을 한 문장으로 반환
품질 기준설명
요약 금지내용 요약이 아닌 구조 분석
원문 인용 필수패턴당 원문 인용 2개 이상
생성 금지텍스트에 없는 내용을 만들지 않음
검증 가능반례 탐색으로 패턴 검증
FINALE

이 스킬은 실패에서 태어났다.
